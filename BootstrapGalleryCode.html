Class four, Printing a Shell

For this class, we could download any creative commons 3D model, and I chose a small shell pot, since it would only take 38 minutes to print, leaving me with plenty of time to set up a larger project later on.



Finished Shell

Finished Shell! Again, pretty stringy on the inside, but it functions well, and the design on the outside is very clean-cut. Another successful print!



Making A Cord Taco

For my big project, I used a Thingiverse taco croc charm and TinkerCAD to create a cord taco, which is a holder for cords. This is a functional build that incorporates my love for tacos.



Ready to Print

This is the finalized model which I used to print. I'm using a filament with multiple different colors blended in, and the whole print should take about 6 hours and be about 7 inches wide and 4 inches tall.



Finished Product!

Featuring a prism pattern on the outside, this cord taco came out just as expected!



Bottom View

Although, I hadn't expected the pattern on the bottom. This is due to the lack of support, though supports aren't necessary for the structure here.
